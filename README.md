# My Giphy Site

Javascript APP that uses the [Giphy API](https://github.com/Giphy/GiphyAPI) to allow users to seach for Random GIF images.

Try the app Here [http://giphy.mariomelchor.com/](http://giphy.mariomelchor.com/)

## Technologies Utilized

1. [jQuery](https://jquery.com/)
2. [Masonry](https://masonry.desandro.com/)
3. [Images Loaded](https://imagesloaded.desandro.com/)
4. [Bootstrap](https://getbootstrap.com/docs/3.3/)
